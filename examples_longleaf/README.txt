Run directories require inputs dir with sequences.csv (similar to AF2 formatting) and slurm running file.
Generating MSA:
Flags must be used in command line. See all input options in get_msa.py (bottom of file)

Running RF2:
Flags must be used in command line. See all input options in run_rf2.py.
Recommended format (for monomers and multimers) is to use -input_dir option - see example in complex_multiple
